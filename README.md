# 파주시 지방재정 투자심사 사전판단·의뢰서 작성 보조 도구

**파주시청 AI 혁신동아리 ACE팀 | v2.2**

## 개요

지방자치단체 공무원이 지방재정 투자심사 업무를 처리할 때 필요한 두 가지 핵심 작업을 자동화하는 웹 도구입니다.

1. **심사 대상·기관 자동 판정** — 사업 유형과 총사업비를 입력하면 자체/경기도/중앙 심사 여부, 면제·비대상 여부를 2025년 행안부 운영기준 근거 조문과 함께 제시
2. **투자심사 의뢰서 초안 자동 작성** — 행안부 서식(1~7장)에 맞춰 입력값 기반 초안 생성, Word(.docx) 다운로드 지원

별도 설치 없이 브라우저에서 HTML 파일을 열어 바로 사용합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 심사기관 판정 | 사업유형·금액·재원 구조에 따라 자체/경기도/중앙 자동 분류 (법령 호·목 근거 표시) |
| 재심사 판단 | 7개 재심사 사유 자동 점검 (500억 특례식, 공제계산 포함) |
| 사업비 산출 | 약식(서울시 가이드라인 단가×연면적) / 세부(37항목 직접 입력) 선택 |
| 사업기간 계산 | 계획수립~준공 12단계 누적 일정 자동 계산 |
| 의뢰서 2장 자동 작성 | 산출방식 연동 — 총사업비 산출내역표 자동 생성 |
| 의뢰서 3장 자동 작성 | 12단계 계산기 연동 — 세부추진계획·일정표 자동 생성 |
| Word 내보내기 | 행안부 서식 순서(1→2→3→4→5→6→7장) .docx 다운로드 |
| AI 연동 | ChatGPT / Gemini / Claude 선택, 서술형 항목 초안 및 규정 Q&A |
| 와이드 모니터 지원 | 1680px 이상에서 AI 채팅 독 3열 레이아웃 자동 전환 |

## 근거 문서

- 지방재정 투자심사 및 타당성조사 운영기준 (행정안전부, 2025.4.30.)
- 지방재정법 시행령 제41조 제2항 (대통령령 제35186호, 2025.1.7.)
- 투자심사·타당성조사 심사규칙 (행정안전부령 제464호)
- 서울시 공공건축물 건립 공사비 책정 가이드라인 (2024, 기술심사담당관)
- 엔지니어링사업대가 기준 별표1 (국토교통부)

## 사용 방법

1. `파주시_투자심사_에이전트_v2.2.html` 파일을 브라우저에서 열기
2. **기본정보** 탭에서 사업 유형·총사업비·재원·산출방식 입력
3. **분석 실행** 버튼 클릭 → 심사기관·면제·재심사 결과 확인
4. **사업비·기간** 탭에서 산출 내역 및 사업일정 입력
5. **의뢰서 초안** 탭에서 자동 생성된 초안 확인 및 Word 다운로드
6. (선택) **AI 설정**에서 API 키 입력 후 서술형 항목 AI 보조 작성

## AI 설정 안내

- 설정 아이콘 → 엔진 선택(GPT/Gemini/Claude) → API 키 입력 후 저장
- API 키는 **사용자 브라우저 localStorage에만 저장**되며 서버로 전송되지 않습니다
- 키 입력칸은 보안상 저장 여부만 표시하며 값을 다시 보여주지 않습니다

## 개발 환경

- 단일 HTML 파일 (서버·DB 없음, 순수 프론트엔드)
- HTML / CSS / JavaScript (Vanilla)
- Word 내보내기: docx.js (CDN)
- 개발: 파주시청 ACE팀 (AI 혁신동아리) — 비전공자 + AI 도구 활용 개발
